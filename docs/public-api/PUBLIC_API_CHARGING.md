# 전기차 충전소 API 명세

## 개요
- **제공**: 한국환경공단
- **서비스명**: 전기자동차 충전소 정보 조회 서비스 (EvCharger)
- **Base URL**: `http://apis.data.go.kr/B552584/EvCharger`

---

## 오퍼레이션

| 오퍼레이션 | 설명 |
|------------|------|
| `getChargerInfo` | 충전기 정보 조회 (위치, 타입, 상태 등) |
| `getChargerStatus` | 충전기 상태 조회 (실시간 상태) |

---

## 1. 충전기 정보 조회 (getChargerInfo)

### 요청
```
GET /B552584/EvCharger/getChargerInfo
```

### 요청 파라미터

| 파라미터 | 필수 | 설명 |
|----------|------|------|
| serviceKey | O | 인증키 (URL Encode) |
| pageNo | O | 페이지 번호 |
| numOfRows | O | 한 페이지 결과 수 (10~9999) |
| dataType | O | 데이터 타입 (JSON, XML) |
| zcode | X | 시도 코드 (행정구역코드 앞 2자리) |
| zscode | X | 지역 상세코드 |
| kind | X | 충전소 구분 코드 |
| statId | X | 충전소 ID |
| chgerId | X | 충전기 ID |

### 응답 예시 (JSON)
```json
{
  "resultCode": "00",
  "resultMsg": "NORMAL SERVICE.",
  "numOfRows": 10,
  "pageNo": 1,
  "totalCount": 15,
  "items": {
    "item": [{
      "statNm": "기후대기관",
      "statId": "28260005",
      "chgerId": "02",
      "chgerType": "03",
      "addr": "인천광역시 서구 환경로 42",
      "lat": "37.569620",
      "lng": "126.641973",
      "stat": "2",
      "output": "50",
      "parkingFree": "Y",
      "busiNm": "한국자동차환경협회"
    }]
  }
}
```

### 주요 응답 필드

| 필드 | 설명 |
|------|------|
| statNm | 충전소명 |
| statId | 충전소 ID |
| chgerId | 충전기 ID |
| chgerType | 충전기 타입 코드 |
| addr | 주소 |
| lat / lng | 위도 / 경도 |
| stat | 충전기 상태 코드 |
| output | 충전용량 (kW) |
| parkingFree | 주차료 무료 여부 (Y/N) |
| busiNm | 운영기관명 |
| useTime | 이용가능시간 |

## 2. 충전기 상태 조회 (getChargerStatus)

### 요청
```
GET /B552584/EvCharger/getChargerStatus
```

### 추가 파라미터

| 파라미터 | 필수 | 설명 |
|----------|------|------|
| period | X | 상태갱신 조회 범위 (분, 기본값 5, 최대 10) |

### 응답 필드

| 필드 | 설명 |
|------|------|
| statId | 충전소 ID |
| chgerId | 충전기 ID |
| stat | 충전기 상태 |
| statUpdDt | 상태 갱신 일시 |

---

## 코드 정의

### 충전기 상태 (stat)

| 코드 | 설명 |
|------|------|
| 0 | 알수없음 |
| 1 | 통신이상 |
| 2 | 사용가능 |
| 3 | 충전중 |
| 4 | 운영중지 |
| 5 | 점검중 |

| 코드 | 설명 |

| 04 | DC콤보 |

| 09 | NACS |
| 10 | DC콤보+NACS |
